# Module 02 – Microsoft Defender XDR & Office 365 Security Policies

## Objective
Configure and apply Microsoft Defender for Office 365 security policies to protect against email-based threats.

## Tools Used
- Microsoft Defender XDR
- Microsoft Defender for Office 365
- Microsoft 365 Security Portal

## What I Did
- Accessed the Microsoft Defender XDR portal
- Configured preset security policies (Standard and Strict protection)
- Applied policies to specific users and groups
- Enabled protections including:
  - Anti-phishing
  - Anti-malware
  - Safe Links
  - Safe Attachments
- Initialized the Microsoft Defender XDR workspace
- Verified successful deployment of security configurations

## Key Configurations
- Standard protection applied to tenant domain
- Strict protection applied to high-risk group (e.g. Leadership)
- Enabled advanced email threat protection features
